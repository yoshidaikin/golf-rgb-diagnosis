# Turf Vision 0｜AR撮影ガイド初号機 Ver.0.0.2

## 新機能
- 「フォト・ファイル」と「AR撮影ガイド」の切替
- カメラ映像を見ながら緑色の撮影範囲枠で位置合わせ
- 中央線・水平目安・撮影注意表示
- 撮影前は診断しない
- 撮影後に「撮り直す／この画像を使用」を選択
- 「この画像を使用」を押した後だけ解析範囲へ進む
- 既存の診断・教師データ・JSONバックアップ・相談用レポート機能を継続

## 利用条件
- GitHub PagesなどHTTPS環境で使用
- 初回のみスマホのカメラ使用許可が必要
- AndroidはChrome、iPhoneはSafariの最新版を推奨
