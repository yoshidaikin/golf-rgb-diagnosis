芝生RGB簡易診断 Ver0.6.8 現場判定調整版

【更新するファイル】
・app.js
・sw.js

【GitHubへの反映】
1. golf-rgb-diagnosis を開きます。
2. Add file → Upload files を選択します。
3. このZIP内の app.js と sw.js をアップロードして上書きします。
4. Commit changes を押します。
5. GitHub Pages の緑色の完了表示を確認します。
6. アプリを開き直します。古い表示が残る場合は、Chromeを完全に閉じて再度開いてください。

【Ver0.6.8の調整内容】
・ベント芝：全面ドライやVARIマイナスを強く評価し、強ドライがBで止まりにくいよう調整。
・コウライ芝：葉先・芝目による細かな点反応を抑え、まとまった反応を中心に表示。
・ティフトン／バミューダ：模様指数だけでC判定にせず、白化候補率・ドライ指数と組み合わせて評価。
・薄目砂／散布直後／厚目砂を選択した場合、模様指数を自動補正。
・異常強調：画像全体ではなく、局所的にまとまった弱い部分だけを半透明表示。
・相談用レポートに白化候補率を追加。

【注意】
この版は今回までの実画像を参考にした現場試験版です。病害・虫害・ドライの原因を確定するものではありません。現地確認と定期比較を併用してください。
