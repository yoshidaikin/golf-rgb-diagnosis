芝生RGB簡易診断 Ver0.6.4 固定基準グラデーション版

【変更点】
・VARIを初期選択に変更
・芝種別の固定色基準を追加
・画像内相対表示との切替を追加
・相談用レポートにも固定／相対の種別を記載
・診断判定、ドライ指数、教師データの保存方式は変更していません

【アップロード】
GitHub の golf-rgb-diagnosis に app.js と sw.js を上書きしてください。
アップロード手順.txt はアプリ動作に不要です。
GitHub Pages が緑になった後、アプリを一度閉じて再度開いてください。
古い表示が残る場合はChromeでページを再読み込みしてください。

【おすすめ設定】
表示方式：数値グラデーション表示
対象：VARI
色の基準：芝種別の固定基準

※固定基準値は現段階の試験基準です。現場画像を確認しながら今後調整します。
